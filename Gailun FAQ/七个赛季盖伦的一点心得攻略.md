# 对线技巧

盖伦在对线大多数上单的时候并没有优势，爆发不如瑞文，剑姬，消耗不如潘森、凯南、璐璐，对拼很难打过诺手，但是我们有自己的信念，合理的利用好自己手中的宝剑，做好每一个细节，盖伦并不会输给任何人。
下面先介绍几个对线的细节。
盖伦目前版本主E副Q是主流，但是有一些加点的细节需要跟大家分享。
1.盖伦加点，一级考虑加W
大多数玩盖伦的人出门就把Q技能点好了，其实这是个不好的习惯，因为很多时候你的Q技能只能帮你去补几个刀，如果你执着的要去Q对面英雄，不但会被反打还会引到小兵的仇恨。在对线一些远程英雄，或者你不准备一级就去消耗对手的话，可以考虑1级的时候加W技能，这样的好处是当你达到3级的时候会比原来多出5点的护甲和魔抗，这在对拼的时候是非常关键的，也是制胜的关键因素。

<br/>

1.盖伦加点，前期加2级W
2级的W，能提供3s的减伤，前期可以考虑牺牲一点E技能来加，E技能是一个成长型的技能，你每升3级会多转1圈，在你没有出AD装备的时候2级的E仅仅比1级的E多20几点的伤害。
在对线会玩的瑞兹的时候，可以考虑2级的W，因为他的连招中会有2个W禁锢，被禁锢的盖伦除了按W没有其他办法，而且不到6级，盖伦也不要想能单杀瑞兹，所以前期2级的W能帮你提高自己的生存能力。
在对线瑞文的时候也可以考虑，前提是你感觉对手玩的比你好，你在前期没有单杀他的机会，会玩的瑞文伤害一套打完是不止2s的，毕竟生存是输出的基础。
2.主Q加点
当你对线一些拥有位移或者是有加速技能的英雄的时候，可以考虑主Q技能，原因是你主了E也不能把E技能的伤害打满，甚至是根本碰不到对手，那么主E就毫无用处了。
Q技能升级除了额外的输出加成之外，额外的加速时间也能帮助你更快的接近对手。
当然有些时候2级的Q就已经够了，具体的情况需要灵活加点。

(主要是针对那些消耗特别高的英雄)

# lol攻击后摇是什么 攻击后摇怎么取消

相信细心的玩家朋友观察lol中一个英雄完成一个完整的动作需要分为两个部分，专业点的称呼将这两个部分分别成为攻击前摇和攻击后摇，那么lol攻击后摇是什么意思呢？lol攻击后摇怎么取消呢？


暗夜猎手——VN

　　我们以暗夜猎手VN举例来为大家分析一下攻击前摇和攻击后摇，假设VN攻速是1.0，也就是没1s攻击1次
　　vn攻击敌人时会选拔出身后的武器然后攻击敌人，大概是0.3s，这个拔出武器然后攻击敌人的动作就是攻击前摇；
　　在攻击完一次后剩下的vn就会用0.7s的时间收起武器，回复到不攻击状态，这段过程就是我们所谓的攻击后摇，这两端动作完成才算是一个英雄普攻完成（强制性的攻击后摇，每个英雄都有）
　　但是有的玩家就会发现职业玩家或者是操作犀利一点的大神他们普攻时动作行云流水，感觉没有收武器（攻击后摇）就可以打第二次，而我们自己玩的时候确需要等攻击后摇结束才可以打第二次，这里就引入了攻击后摇取消的概念。


鳄鱼利用九头蛇取消攻击后摇


　　首先大家要明确一点：攻击后摇是不可以取消的，但是可以减少后摇时间
　　方法一：利用走位取消攻击后摇——在普攻第一次完成攻击前摇的时候，当你的英雄准备收武器（攻击后摇）移动自己的英雄就可以很快速的再次打出第二次普攻
　　分析：这里是利用移动来减少攻击后摇时间，这个方法成功的标志就是你的英雄在A完人后武器没收回去的时候就打出第二下，这就证明你成功了
　　方法二：利用技能取消攻击后摇
　　这个和利用移动取消攻击后摇的方法一致，不过换成了在英雄攻击后摇开始时使用特定技能来减少攻击后摇时间，从而达到减少攻击后摇的延迟感。
　　分析：原理和方法一一致，这里的特定技能都是一些瞬发型技能，例如vn的Q技能，瑞文的Q技能甚至是闪现等技能。
　　方法三：利用特定装备取消攻击后摇
　　分析：一般都是九头蛇，这件装备使用后英雄可以迅速攻击，从而达到李颖装备完成取消攻击后摇的目的。

# 兵线控制
上单，顾名思义你是一个人在战斗，除了打野来帮你，你可能忘了你身边会有5个友方小兵一直在默默的帮助你，如何利用好这几个小兵也是上路对拼的关键细节。
作为近战的盖伦，当你用普攻或者是Q技能打对方的时候，势必对方的小兵会打你，而你的小兵正在打对方小兵，对面的小兵血量就会比你的小兵多，所以兵线就会往对面塔下推。说的简单一点就是谁打对手，谁的兵线就会往对面推，这也是为什么盖伦经常会被人推兵线，因为对面会经常A你。
作为前期线上并不强势的盖伦，可以让兵线尽可能的靠在自己的塔附近，最理想的情况是正好对面的近战兵不会被你的塔A到，2级以后补兵尽量只A最后一刀，如果需要A+Q去消耗对手，Q完以后马上进草拉掉小兵仇恨。
2.技能的正确使用
上文提到过兵线控制大家务必多看几遍，盖伦在对拼的时候的一些基本手法有AQ，用Q取消A的后摇，瞬间打出2个伤害，然后通过Q技能给你提供的额外移速远离对手或进草，避免被反消耗，也避免被对面的小兵打更多的血，也避免自己的兵线往对面推。
3级以后，如果要打，大多数时候盖伦是用E技能起手的，转完一套后+AQ，这是最理想的情况，如果你先用Q技能起手，当你Q他的时候加上你一个E技能的时间对方的小兵会一直在打你，4秒多的时间对面小兵给你造成的伤害大概是对面英雄A你2下的伤害，是非常致命的，所以不是特殊情况，请不要用Q起手。
不过Q起手也有他的好处，在于对手会被沉默1.5s，这段时间他不能使用技能对你造成伤害，也对打乱他技能的节奏，如对线瑞文的时候，你要打他就必须要用Q起手，不然你根本摸不到他。
总结一下就是尽量不让对面的兵A你的前提下造成更多的伤害，对拼的时候不要无脑用技能，可以等你的对手把防御技能用完之后再用出你的伤害技能。
